# Tipos de dados

Conforme o ECAMScript standart temos 9 tipos de dados:

* Data types
   * Primitive / Primitive valvue
   * Strutural
   * Strutural Primitive

## Primitivos

* String "ABC" "BCA" "bca" "Bca"
* Number
* Boolean
* undefined
* Symbol
*BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...
* Function
